#### branch 상태 확인

    git branch
    git branch -v # 마지막 커밋내용
    git branch -r # 리모트 저장소
    git branch -a # 로컬, 리모트 모두

#### brach 만들기

    git branch <branch>

#### branch 이동하기

    git checkout <branch>

    git checkout -b <branch> # 생성과 동시에 이동

#### branch 삭제하기

    git branch -d <branch>

#### branch 이름 변경

    git branch -m <branch1> <branch2>
